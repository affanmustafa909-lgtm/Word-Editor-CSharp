# Word Editor

Word Editor is a simple and user-friendly text editing application developed using C# and Windows Forms in Microsoft Visual Studio. The application provides basic word processing features such as creating, editing, formatting, saving, and opening text documents.

---

# Features

- Create New Documents
- Open Existing Files
- Save and Save As Functionality
- Rich Text Format (.rtf) Support
- Plain Text (.txt) Support
- Word Count Tracking
- Copy, Cut, and Paste Operations
- Undo Functionality
- Bold, Italic, and Underline Text Formatting
- Toolbar Shortcuts
- User-Friendly Interface
